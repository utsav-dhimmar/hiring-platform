# HR Platform Setup Guide

This project is split into two main parts: the backend and the frontend. Please follow the instructions in each directory's setup guide to get the full platform running.

## Components

### 1. [Backend Setup Guide](backend/setup.md)
The backend is built with Python (FastAPI/SQLAlchemy) and requires a PostgreSQL database. 
- API service
- Database migrations
- Async workers

### 2. [Frontend Setup Guide](frontend/SETUP.md)
The frontend is built with React, Vite, and Bun/npm.
- Development server
- UI components
- State management

## Quick Start (Summary)

1. **Backend**: Navigate to `/backend`, set up your virtual environment, and run the server.
2. **Frontend**: Navigate to `/frontend`, install dependencies, and start the development server.

Detailed instructions can be found in the links above.
